Clearvo E-Invoicing

Submit invoices to national tax authorities automatically — the moment you post in Odoo. One integration, every mandate.

Connects in under 2 minutes. Create a Clearvo account, paste your API key, done. No XML configuration, no format mapping, no per-country setup.

Supported countries

France (Factur-X) Germany (XRechnung) Belgium (Peppol) Italy (FatturaPA) Poland (KSeF) Spain (VeriFactu) Portugal (SAFT-PT) Romania (e-Factura) Hungary (NAV) Greece (myDATA) Netherlands Austria Norway + more

How it works

  • Auto-submit on post: when you confirm an invoice, Clearvo receives it, generates the correct XML format, and submits it to the relevant authority.
  • Live status: every invoice shows its clearance status — Pending, Accepted, Rejected — directly in Odoo.
  • Manual retry: if a submission fails, a Retry button appears on the invoice with the error detail.
  • Multi-company safe: each company has its own API key and auto-submit toggle.
  • Sandbox mode: use a csk_test_ key for testing — no invoices reach tax authorities.

Tax code configuration (required)

Clearvo uses EN16931 tax category codes (S, AA, AE, K, G, E…) to determine VAT treatment per invoice line. You must set the Clearvo Tax Code field on every Odoo tax you use — go to Accounting → Configuration → Taxes and open each tax to set it. Common values: S (standard rate), AA (reduced rate), AE (reverse charge), K (intra-EU zero-rated). The module will surface a clear error if a tax is missing its code.
