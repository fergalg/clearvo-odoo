Clearvo E-Invoicing

Submit invoices to national tax authorities automatically — the moment you post in Odoo.

Connects in under 2 minutes. Create a Clearvo account, paste your API key into Odoo, and you're done. No XML configuration, no format mapping, no per-country setup required.

How to get started

1
Create a Clearvo account
Sign up free at clearvo.io. No credit card needed.
2
Generate an API key
In the Clearvo dashboard go to Settings → API Keys and create an entity key.
3
Connect in Odoo
Go to Accounting → Configuration → Clearvo E-Invoicing and paste the key.

What it does

  • Auto-submit on post: when you confirm an invoice, Clearvo receives it, generates the correct XML format (UBL, FatturaPA, XRechnung, etc.), and submits it to the relevant tax authority.
  • Live clearance status: every invoice shows its status — Pending, Accepted, Rejected — directly on the invoice form in Odoo.
  • Push notifications: Clearvo sends a webhook when a status changes, so Odoo updates in real time without polling.
  • Manual retry: if a submission fails, a Retry button appears with the full error detail from the authority.
  • Multi-company: each Odoo company has its own API key, currency, and auto-submit toggle.
  • Sandbox mode: use a csk_test_ key to test the full flow — no invoices reach any tax authority.

Tax codes — automatic, with manual override

Clearvo uses EN16931 tax category codes (S, AA, AE, K, G, E, O…) to determine VAT treatment per invoice line. You don't need to configure anything — the module detects the correct code automatically from your Odoo tax rate and fiscal position tags:

Your tax in OdooAuto-detected code
Any tax with amount > 0%S — Standard rate
0% with intra-EU / IC tagK — Intra-EU supply
0% with export / outside EU tagG — Export
0% with reverse charge tagAE — Reverse charge
0% with no special tagE — Exempt
Fixed amount taxO — Out of scope

If auto-detection isn't right for a specific tax, open the tax in Accounting → Configuration → Taxes and set the Clearvo Tax Code Override field. The override takes precedence; the auto-detected code is shown next to it so you can compare.

Requirements

  • Odoo 17.0
  • The account module (Accounting)
  • A Clearvo account — free trial at clearvo.io
  • Internet access from the Odoo server (outbound HTTPS to api.clearvo.io)
Private / on-premises instances: if your Odoo server is not reachable from the public internet, real-time webhook push won't work. Enable the Clearvo: poll pending invoices scheduled action (under Technical → Scheduled Actions) to poll for status updates instead.
